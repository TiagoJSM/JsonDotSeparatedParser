# JSON config Parser 

This is an application that parses json files and merges them to form a configuration file.

## Prerequesites

Requires installation of .net core toolchain to run and build the application.

## Install Dependencies

```bash
dotnet restore
```

## Build

```bash
dotnet build
```

## Run Tests

```bash
dotnet test
```

## Run App

```bash
cd Parser
dotnet run
```
